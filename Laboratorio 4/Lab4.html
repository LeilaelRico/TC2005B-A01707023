¿Qué diferencias y semejanzas hay entre Java y JavaScript?

En base a ISDI Coders, una de las principales diferencias entre estos dos lenguajes es que "Java crea aplicaciones que tras ser compiladas (traducidas a código ejecutable) se pueden ejecutar en navegadores a través de una máquina virtual, mientras que Javascript se ejecuta nativamente en los navegadores sin necesidad de ninguna máquina virtual."

¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)

  • Date.now(): Devuelve el valor numérico correspondiente a la hora actual.
  • Date.parse(): Transforma una cadena que representa una fecha, y devuelve el número de milisegundos desde el 1 de enero de 1970, hora local 00:00:00.
  • Date.UTC(): Acepta los mismos parámetros que la forma más larga del constructor, y devuelve el número de milisegundos en un objeto Date desde el 1 de enero de 1970, hora universal 00:00:00.
  • Date.prototype.getFullYear(): Devuelve el año de la fecha indicada acorde a la hora local.
  • Date.prototype.setFullYear(): Fija el año completo para una fecha específica de acuerdo a la zona horaria local. Devuelve un nuevo valor de tipo timestamp.
Fuente: Developer Mozilla.

¿Qué métodos tienen los arreglos? (Menciona al menos 5*)

  • Array(): Crea un nuevo objeto Array.
  • get Array[@@species]: La función del constructor se utiliza para crear objetos derivados.
  • Array.from(): Crea una nueva instancia de Array a partir de similarAUnArray, un objeto iterable o parecido a un array.
  • Array.isArray(): Devuelve true si valor es un array, y false en caso contrario.
  • Array.of(): Crea una nueva instancia de Array con un número variable de parámetros, independientemente del número y del tipo de dichos parámetros.
  • Array.prototype.fill(): Asigna un valor estático a todos los elementos del array entre las posiciones inicio y fin.
  • Array.prototype.pop(): Elimina el último elemento de un array, y devuelve dicho elemento.
Fuente: Developer Mozilla.
