Respuesta a preguntas

Como ingeniero de software ¿cuál es tu recomendación sobre el uso de !important en un CSS?

En caso de que se trabajen con diferentes estilos y el navegador soporte esa función, "!important" puede ser utilizado para definir aspectos visuales específicos en las páginas.
Fuente: Declaración !important en CSS

Si se pone una imagen de fondo en una página HTML, ¿por qué debe escogerse con cuidado?

Hablando desde el estilo visual, creo que debe de elegirse teniendo en cuenta el texto que pueda haber en la página, especialmente para que este pueda seguirse visualizando fácilmente y evitando que se pierda con la imagen o que el mismo fondo pierda calidad en caso de que esta deba de estirarse demasiado.

Como ingeniero de software, ¿cuál es tu recomendación al elegir las unidades de un propiedad de estilo entre %, px y pt?

Dependerá de lo que se quiera lograr:
  • "%": Porcentaje relativo al elemento contenedor; yo la utilizaría más para definir el tamaño de imágenes o elementos "grandes".
  • "px": Unidad basada en los pixeles de un dispositivo, creo que se utiliza más para definir el tamaño de las fuentes principalmente.
  • "pt": Unidad de punto, en base a w3.org, se buscaría utilizar en caso de diseñar una página que busque imprimirse para lograr que el estilo buscado se rerspete.
Fuente: EM, PX, PT, CM, IN…

¿Por qué el uso de una versión minimizada del CSS mejora el rendimiento del sitio?

En base a lo visto en clase, porque, como no se tratan de instrucciones que deben de compilarse, el navegador tiene que interpretar toda la información línea por línea.
